# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-11

### Added
- **Local Document Importing**: Import PDFs and images directly from the local file system.
- **Batch Sequential Import**: Coordinated queue that imports multiple PDF or image files sequentially, preventing memory spikes.
- **Document Scanning**: High-quality physical document scanning utilizing Google ML Kit.
- **Multi-page Scanning**: Drag-and-drop page reordering with stable keys, page deletion, page replacement, and custom PDF document naming.
- **Built-in Document Viewers**: Interactive PDF viewer and stable centered fitted image viewer with focal-point pinch zoom and double tap gesture handlers.
- **File Management**: Bulk select mode on Home and Recycle Bin list/grid layouts.
- **Recycle Bin (Soft Delete)**: Recycle Bin directory supporting soft deletes, folder/selection restores, and permanent purges.
- **Dynamic Localization**: Fully localized application translations supporting English and Tamil (தமிழ்).
- **Material 3 Themes**: System-adaptive light and dark color schemes.
- **Responsive Layout**: Shell navigation drawer adapting between mobile and tablet layout widths.
